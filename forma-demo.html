01

Layout primitives.
Not utility soup.

Tailwind gives you thousands of atomic classes. FORMA gives you the 12 layout patterns that actually cover 95% of real sites — designed to compose, not conflict.

⏸
.stack
Vertical rhythm via gap, not margin. The foundation of every layout.
↔
.cluster
Inline grouping that wraps. Nav links, tag lists, button groups.
▦
.grid
Responsive CSS grid with data-cols. No breakpoint arithmetic.
◫
.sidebar
Asymmetric two-column. Collapses to one column on mobile.
⊞
.cover
Full-viewport hero sections. The place-items:center hero that just works.
⬚
.frame
Aspect-ratio containers for images and video. Always perfect ratio.
02

Motion that means something.

Eight motion primitives via data attributes. No JavaScript required from you. Add data-reveal to any element and it animates on scroll.

Attribute
data-reveal fade + slide up
data-reveal="left" slide from left
data-reveal="clip" clip-path wipe
data-stagger stagger children
data-parallax="0.3" scroll depth
data-magnetic cursor attraction
data-split-text word-by-word reveal
data-counter number animation
↑

IntersectionObserver powers all reveals. No scroll listeners. No layout thrash. Smooth on 60fps.
